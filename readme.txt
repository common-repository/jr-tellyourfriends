== Description ==

This plugin allows your users to tell their friends about your website - With a simple form in the sidebar!

You can also set a limit of the amount of friends each visitor can recommend, just in case you have any worries about the amount of e-mails you'll send out.

Just enable the plugin, enable the widget in the Appearance --> Widgets menu and then change the settings in Settings --> JR TellYourFriends!

== Installation ==

1. Upload `/jr-tellyourfriends.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR TellYourFriends Settings in your Administration Panel, Settings --> JR TellYourFriends.
4. Enable the widget in Appearance --> Widgets.
